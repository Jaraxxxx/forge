## Overview

Industrial precision meets developer ergonomics. Forge is a professional AI coding harness.
The UI conveys trust, speed, and capability — like a precision instrument, not a toy.

Dark zinc base with indigo accent. High contrast for readability. Monospace for data.
Every pixel serves purpose. No decoration — only information density and clarity.

## Colors

- **Base (#09090b):** Deep zinc background — reduces eye strain during long sessions
- **Surface (#121215):** Panel backgrounds — sidebar, cards, input areas
- **Elevated (#18181b):** Hover states, dropdowns, modals
- **Border (#27272a):** Separators, card borders — subtle but visible
- **Accent (#6366f1):** Primary interaction — buttons, links, active indicators

## Components

Status bar is thin (32px), fixed at bottom. Shows model, tokens, context bar.
Model selector is a dropdown in the status bar. 
Tool cards are collapsible — header shows name + status, body shows result.
Messages use prose with monospace code blocks.
